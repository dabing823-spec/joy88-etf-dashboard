# 報告章節 Markdown 模板

每個章節的完整模板。報告引擎根據報告類型和 Mode 條件式組裝。

---

## Ch.1 市場總覽

```markdown
## 1️⃣ 市場總覽

📊 **加權指數**：{收盤價/現價} （{漲跌點數} / {漲跌%}）
📈 **成交量**：{成交金額}億（vs 5日均量 {5日均量}億）
🌏 **國際連動**：
- 美股：道瓊 {漲跌%} / 那斯達克 {漲跌%} / S&P500 {漲跌%}
- 費半：{漲跌%}
- 亞股：日經 {漲跌%} / 韓股 {漲跌%}

🏷️ **市場定調**：{多頭📈/震盪📊/空頭📉/事件驅動⚡} — {一句話說明}
```

## Ch.2 重大新聞與事件

### 盤後/週報/月報版本（完整）
```markdown
## 2️⃣ 重大新聞與事件

### 📰 {新聞標題 1}
- ✅ 事實：{一句話描述}
- 🔶 影響研判：{對市場/特定標的的推論，附依據}

### 📰 {新聞標題 2}
- ✅ 事實：{一句話描述}
- 🔶 影響研判：{推論}

{重複 2-5 則，依重要性排序}

📅 **今日/本週關注事件**：
- {時間} — {事件}（影響標的：{XXX}）
- {時間} — {事件}
```

### 盤前日報版本（精簡，最多 3 則 × 2 行）
```markdown
## 2️⃣ 重大新聞

📰 **{新聞標題 1}**：✅ {一句話事實} → 🔶 {一句話推論}
📰 **{新聞標題 2}**：✅ {一句話事實} → 🔶 {一句話推論}
📰 **{新聞標題 3}**：✅ {一句話事實} → 🔶 {一句話推論}

📅 今日關注：{事件1}（{時間}）| {事件2}（{時間}）
```

盤前日報的新聞選擇原則：只選對今日開盤最有直接影響的 1-3 則，捨棄背景性新聞。

## Ch.3 技術面信號

### 盤後/週報/月報版本（完整）
```markdown
## 3️⃣ 技術面信號

📐 **加權指數技術面**：
- 均線：5MA {值}({上/下}) / 10MA {值} / 20MA {值}({上/下}) / 60MA {值}
- 指數位階：{5MA 之上 → 強勢 / 5MA-20MA 間 → 中性 / 20MA 之下 → 弱勢}

📊 **動能指標**：
- RSI(14)：{值} — {超買(>70)/中性/超賣(<30)}
- KD(9,3,3)：K={值} D={值} — {黃金交叉/死亡交叉/中性}
- MACD：DIF={值} MACD={值} 柱狀={值} — {多方/空方/收斂}

🔑 **關鍵價位**：
- 壓力：{價位1}（{原因}）/ {價位2}
- 支撐：{價位1}（{原因}）/ {價位2}

🔶 **技術面研判**：{綜合判斷，標註為推論}
```

### 盤前日報版本（精簡，純數值 + 關鍵價位）
```markdown
## 3️⃣ 技術面

📐 收盤 {值} | 5MA {值} | 20MA {值} | 60MA {值} | 位階：{強/中/弱}
📊 RSI {值} | K/D {值}/{值} | MACD 柱 {正/負}
🔑 壓力 {值} / {值} ─ 支撐 {值} / {值}
```

注意：盤前版不寫解釋段落，只列數據。交易者看數字就懂。

## Ch.7 風控燈號（Mode A Only）

```markdown
## 7️⃣ Joy88 風控燈號

🚦 **三系統狀態**：
- HWM 回撤：{%} → {🟢正常/🟡預警/🔴一級/🔴🔴二級}
- VIX 燈號：{值} → {🟢/🟡/🔴}
- 指數位階：{強勢/中性/弱勢} → 係數 {1.2/0.9/0.6}

📐 **槓桿狀態**：
- 目標槓桿：{值}x
- 實際槓桿：{值}x（{需降槓/合理範圍/可加碼}）

⚡ **鐵律檢查**：
- 風險指標 <115%？→ {是/否}（觸發則強制減倉）
- 空單禁砍規則？→ {適用/不適用}

💡 如需完整風控運算（含砍倉序列與崩跌試算），請使用 position-tracker。
```

注意：此章節是 position-tracker 的精簡摘要版。
完整數據需要：權益數、HWM、VIX 期貨收盤、指數相對均線位置。
缺少任何一項時，標註缺失並建議用戶提供或執行 position-tracker。

### 月報變體（Ch.8 → 下月展望）
月報的第 8 章改為更長期的展望，含：
- 月度績效回顧（Mode A）
- 下月重大事件日曆
- 策略方向調整建議
- 總經象限研判（通膨/成長矩陣）
